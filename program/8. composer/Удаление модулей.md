1. ***Удаления пакета:***


> [!NOTE] Что произойдет
>  1. Пакет будет удален из файла composer.json.
>  2. Пакет будет удален из файла composer.lock
>  3. Файлы пакета будут физически удалены из директории vendor/.
>  4. Автозагрузчик (autoload) будет обновлен.

*Содержимое файла composer.json до удаления существующего модуля:*

![[Снимок экрана от 2026-05-15 10-41-26.png]]

*Удаление модуля например psr/log (удаление выполняется без указания версии):*

```
sudo -u www-data composer remove psr/log
```
