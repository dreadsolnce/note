
#### Зайти на сайт openrouter.ai

Получить api ключ вида `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`

#### Выбрать бесплатную модель

Например модель Qwen: Qwen3 Coder 480B A35B (free)

Скопировать ее название: `qwen/qwen3-coder:free`

#### Настроить подключение в qwen cli

Подключить провайдера > Third-party Providers > OpenRouter 

Ввести api key: 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

Ввести название модели: qwen/qwen3-coder:free

Ввести (при необходимости) endpoint подключения к api openrouter: https://openrouter.ai/api/v1

Путь с настройками подключения находится по адресу: ~/.qwen/settings
