
***Выполняем авторизацию с помощью google***

`1. Login with Google`


> [!NOTE] Информация
> Если отсутствует графический интерфейс то необходимо перейти по ссылке которая отображается в консоли


> [!NOTE] Ошибка
> Если возникает ошибка при аутентификации через google аккаунт, что типа необходимо добавить переменную окружения, то для этого необходимо перейти в google cloud https://console.cloud.google.com и создать какой нибудь проект и в проекте скопировать Project ID. После запуска gemini и ввода какого нибудь promt может потребоваться подключить к проекту api gemini перейдя по ссылке из строки с ошибкой
>
