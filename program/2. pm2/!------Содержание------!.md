
> [!NOTE] Описание
> **PM2** — это менеджер процессов с открытым исходным кодом, предназначенный в первую очередь для приложений ***Node.js***, который позволяет запускать, останавливать, перезапускать и управлять ими, обеспечивая их работу 24/7. Он действует как демон, который автоматически перезапускает приложение при сбое, следит за логами и ресурсами, а также позволяет распределять нагрузку между ядрами процессора.
> Помимо приложений ***Node.js** позволяет запускать и другие приложения например ***Python, Java и другие***.
