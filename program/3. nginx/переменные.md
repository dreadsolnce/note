***$document_root*** ^fd0c13


> [!NOTE] Описание переменной
> Переменная `$document_root` в Nginx содержит абсолютный путь к корневой директории сайта, определённой директивой `root` или `alias` для текущего запроса. Она используется для формирования полного пути к запрашиваемому файлу (обычно в связке с PHP-FPM как `$document_root$fastcgi_script_name`)

**Основные факты о `$document_root`:**

- **Значение:** Соответствует значению `root` в блоках `server` или `location`.
- **Использование:** Чаще всего применяется в директиве `fastcgi_param SCRIPT_FILENAME` для указания PHP-FPM, где именно лежит файл.
- **Особенность с alias:** Если внутри `location` используется директива `alias`, то `$document_root` будет отражать значение `alias`.

**Пример использования:**

```
server {
    root /var/www/html;
    location / {
        # $document_root здесь равен /var/www/html
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
    }
}
```

***$args***


> [!NOTE] Описание переменной
> Переменная $args в Nginx содержит строку аргументов (query string) из URL запроса. Это всё, что идет в адресе после знака вопроса ?.

**Как это работает (на примерах):**

   1. URL: example.com/search
       * $args — пустая строка.
   2. URL: example.com/search?q=apple
       * $args — q=apple
   3. URL: example.com/index.php?category=books&id=123
       * $args — category=books&id=123


**Ключевые особенности:**

   * **Изменяемость: Вы можете менять значение $args прямо в конфиге. Если вы измените $args, Nginx автоматически обновит и $query_string.**

   1     set $args "id=5"; # Принудительно меняем аргументы
   * **Связь с $query_string: В Nginx переменные $args и $query_string абсолютно идентичны и взаимозаменяемы.**
   
   * **Использование в rewrite: Часто используется, чтобы сохранить параметры при перенаправлении:**

   1     # Перенаправит на новый домен, сохранив все параметры (?q=apple и т.д.)
   2     rewrite ^/old-path/(.*)$ /new-path/$1 permanent; 
   3     # (Здесь Nginx добавит $args автоматически, если не поставить '?' в конце)
