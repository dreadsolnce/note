Консольный браузер:

`cd ~

`wget https://github.com/browsh-org/browsh/releases/download/v1.8.0/browsh_1.8.0_linux_amd64.deb

`sudo apt install ./browsh_1.8.0_linux_amd64.deb

`browsh

`rm browsh_1.8.0_linux_amd64.deb

Ввод адреса: Ctrl + L
Выход: Ctrl + Q
