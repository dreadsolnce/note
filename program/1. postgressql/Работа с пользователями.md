#### Создание пользователя
 
 1. **Создание пользователя с паролем**
```
CREATE USER < имя нового пользователя > WITH PASSWORD '< пароль >';
```

 2. **Добавление описания к пользователю**
```
COMMENT ON ROLE my_new_user IS 'Описание: пользователь для приложения аналитики'; 
```
 
#### Просмотр пользователей

- **Подробный вывод с описанием и ролями**
```
\du+
```

- **Вывод только пользователей**
```
SELECT usename FROM pg_user;
```

#### Изменить пароль пользователя:

```
ALTER USER < имя пользователя > WITH PASSWORD '< новый пароль >';
```

#### Удаление пользователя:

```
DROP USER IF EXISTS < имя пользователя >;
```

#### Создание роли Superuser для пользователя:

```
ALTER ROLE tab300_user SUPERUSER;
```

#### Удаление роли Create DB у пользователя:

```
ALTER ROLE tab300_user NOCREATEDB;
```

> [!NOTE] Title
> Краткая справка по другим правам (если нужно):
> * `NOSUPERUSER` — забрать права суперпользователя.
> * `NOCREATEROLE` — забрать право создавать другие роли/пользователей.
> * `NOINHERIT` — запретить наследование прав групп.
> * `NOLOGIN` — запретить пользователю вообще подключаться к базе (заблокировать).
