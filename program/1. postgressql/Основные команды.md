Выполняем вход под пользователем postgres

```
sudo -u postgres -i
```

```
psql
```

Установленная версия postgres:

- psql --version       (из под пользователя postgres)

- SELECT version();    (через psql)
- SHOW server_version; (через psql)

Создание пользователя:

- CREATE USER dbusernametest WITH PASSWORD '12345678';

Просмотр пользователей:

- SELECT usename FROM pg_user;

Создание пользователя:

- CREATE USER my_new_user WITH PASSWORD 'secure_password';

Изменить пароль пользователя:

- ALTER USER имя_пользователя WITH PASSWORD 'новый_пароль';

Удаление пользователя:

- DROP USER IF EXISTS dbusernametest;

Просмотр ролей (атрибутов) пользователей:

- \du+

Просмотр баз данных:

- \l

Подключение к базе данных:
```
\с dbtest
```

Подключение к базе данных из под пользователя базы данных:

```
psql -h localhost -U db_user -d ai 
```

Удаление базы данных

- DROP DATABASE IF EXISTS < Имя базы данных >;

Просмотр владельца схемы public (подключенной базы данных)

- \dn

Просмотр прав доступа (разрешения) на таблицы:

- \z public.*

Просмотр таблиц:

- \dt+

Просмотр версии postgresql:

- sudo -u zabbix psql -h 127.0.0.1 -U zbx_monitor -d postgres -c "SELECT version();"
