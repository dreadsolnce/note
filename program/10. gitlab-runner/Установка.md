# AstraLinux 1.8

***Установка из стандартного репозитория astra linux***

```
sudo apt install gitlab-runner
```

```
gitlab-runner status
```

```
sudo useradd --system --comment "GitLab Runner" --create-home --home-dir /home/gitlab-runner --shell /bin/bash gitlab-runner
```

```
ls /home/
```

```
sudo chown -R gitlab-runner:gitlab-runner /home/gitlab-runner
```

```
sudo chmod 755 /home/gitlab-runner
```

```
sudo gitlab-runner install --user=gitlab-runner --working-directory=/home/gitlab-runner
```

```
sudo gitlab-runner status
```

```
sudo gitlab-runner start
```

```
sudo gitlab-runner status
```

```
sudo systemctl enable gitlab-runner
```

```
id gitlab-runner
```

```
sudo systemctl status gitlab-runner
```

```
sudo gitlab-runner list
```

```
sudo visudo /etc/sudoers.d/gitlab-runner
```

```
+++ 
gitlab-runner	ALL=(ALL:ALL)	NOPASSWD: ALL
```

```
sudo pdpl-user -i 63 gitlab-runner
```

```
sudo -u gitlab-runner -i
```

```
sudo systemctl status gitlab-runner.service
```

```
exit
```
