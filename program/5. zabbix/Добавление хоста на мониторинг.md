***Общий вывод конфигурационного файла /etc/zabbix/zabbix_agent2.conf***

```
cat /etc/zabbix/zabbix_agent2.conf | grep -vP "^\s*$" | grep -v '#'
```


> [!NOTE] 
> Host name как указано на клиенте в файле /etc/zabbix/zabbix_agent2.conf

```
cat /etc/zabbix/zabbix_agent2.conf | grep -vP "^\s*$" | grep -v '#' | grep -P ^Hostname*
```

> [!NOTE]

> PSK -------- : содержимое файла указанное в параметре TLSPSKFile

PSK identity:
```
cat /etc/zabbix/zabbix_agent2.conf | grep -vP "^\s*$" | grep -v '#' | grep -P ^TLSPSKIdentity*
```

![[Снимок экрана от 2026-01-28 12-26-35.png]]

PSK:
```
cat /etc/zabbix/zabbix_agent2.conf | grep -vP "^\s*$" | grep -v '#' | grep -P ^TLSPSKFile* | awk -F= '{print $2}' | xargs cat
```
