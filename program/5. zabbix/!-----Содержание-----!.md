> [!NOTE] Общие замечания
> ***Настройка пользователя zabbix для astra linux***
> ```
> sudo pdpl-user -i 63 zabbix
> ```
> ```
> sudo chown -R zabbix:zabbix /etc/zabbix/zabbix_agent2.d/
> ```
> ***Локальный запуск (проверка) работы метрики (команды):***
> ```
> cd /etc/zabbix/ && sudo -u zabbix zabbix_agent2 -t proc.num[php-fpm]
> ```

> [!important] ВАЖНО
> При наличии ошибки **zabbix_agent2 [3259741]: ERROR: Cannot read configuration: cannot include "zabbix_agent2.d/plugins.d/*.conf": stat .: permission denied** это означает, что у текущего пользователя нет прав на чтение текущей директории. 
> Необходимо запустить команду из директории /etc/zabbix, где предварительно даны права пользователю zabbix
