__Скрипт показывает объем потребляемой памяти процессами:__

__Установка и запуск:

```bash
cd ~ &&
wget https://raw.githubusercontent.com/pixelb/ps_mem/master/ps_mem.py &&
chmod a+x ps_mem.py &&
sudo python3 ps_mem.py 
```
