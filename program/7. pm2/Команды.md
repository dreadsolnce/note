

основная команда для работы с pm2:
```
pm2 status
```

Основная команда запуска добавление нового процесса (из директории где ecosystem.config.js)
```
pm2 start ecosystem.config.js --only НАЗВАНИЕ_ПРОЦЕССА
```

```
pm2 start ecosystem.config.js --only StopFakeConsumer
```

> [!important] Важно
> исполнение pm2 start ecosystem.config.js приводит к тому что все процессы будут запущены (на дев такое делать нельзя, потому что зачастую и дев и прод имеют общий внешний сервис и могут друг другу мешать)

Запуск с передачей переменных окружения:
```
sudo -E -u user /usr/local/bin/pm2 start 2
```

перезапустить процесс
```
pm2 restart НОМЕР_ПРОЦЕССА
```

остановить
```
pm2 stop НОМЕР_ПРОЦЕССА
```

удалить процесс из списка:
```
pm2 delete НОМЕР_ПРОЦЕССА
```

стартовать
```
pm2 start НОМЕР_ПРОЦЕССА
```

логи процесса
```
pm2 logs НОМЕР_ПРОЦЕССА
```

логи ошибок процесса

```
pm2 logs rabbit-dash.photo --err --lines 500
```

подробная информация о процессе:
```
pm2 show 0
```
