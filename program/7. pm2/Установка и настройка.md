***Установка***

***Настройка***

```
sudo mkdir /opt/pm2daemon/
```

Для каждого пользователя создать общую переменную окружения

```
vim ~/.bashrc
```

```+++
export PM2_HOME='/opt/pm2daemon/'
```

Создать доп группу

```
sudo groupadd pm2users
```

```
sudo usermod -aG pm2users kolchinvll
```

```
sudo chown -R :pm2users /opt/pm2daemon/
```

```
sudo chmod -R 775 /opt/pm2daemon/
```

-# Чтобы новые файлы наследовали группу 
```
sudo chmod g+s /opt/pm2daemon/
```
