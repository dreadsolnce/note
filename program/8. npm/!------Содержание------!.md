
> [!NOTE] ОПИСАНИЕ
> **Это паспорт вашего Node.js-проекта.**
> - Хранит список всех библиотек, которые нужны проекту. Любой разработчик может склонировать репозиторий, выполнить  npm install  — и все нужные пакеты установятся автоматически. Без этого файла пришлось бы вручную искать и скачивать каждую библиотеку
> - Определяет как запускать проект, тесты, сборку и т.д. Вместо того чтобы помнить длинные команды, вся команда использует единые:
> 	
> 	- `npm start`          *запустить проект*
> 	- npm test           запустить тесты
> 	- npm run build      собрать продакшен-версию
> 	  
> - Фиксирует какие версии библиотек использовать. Это гарантирует, что проект соберётся одинаково на вашей машине, у коллеги и в CI-pipeline.
