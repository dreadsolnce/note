
```
regexp: '^(.+)$'  
replace: '# \1'
```

> [!NOTE] Описание кода
> - ***replace: '# \1'***: # \1 означает **"вставить то, что было захвачено в первой группе"** из вашего регулярного выражения (`regexp`) на место найденного шаблона.
> - ***regexp: '^(.+)$':*** исходя из того что написано ниже, получается что в первую группу попадает вся строка от начала `^` и до конца `\$`


> [!important] Описание что значит ()
> В регулярных выражениях `(.+)` означает **захват (группировка) любой последовательности из одного или более любых символов**. Здесь `.` — любой символ, `+` — «один или более раз» (квантификатор), а скобки `()` — создание захватывающей группы


```
regexp: '^#'  
replace: '' # Заменяем # на пустоту
```

> [!NOTE] Описание
> Заменяем строки начинающиеся с комментария (символ решетки) на аналогичную строку без комментария. Удаляем символ # в начале каждой строки.


```
regexp: '^#(.*https://download.astralinux.ru.*)$'  
replace: '\1'
```


> [!NOTE] Описание
> Заменяем строку начинающуюся с # и содержащую в себе определенную строку на ту же самую строку только без комментария.

```
regexp: '^deb (.*cdrom.*)$'
line: '# deb \1'
```


> [!NOTE] Описание
> Комментируем строки начинающиеся с ключевого слова deb и содержащие в себе слово cdrom
