***Добавление debian репозитория***

`debian 12 - Astra Linux 1.8`

```
echo "deb https://deb.debian.org/debian/ bookworm main contrib non-free non-free-firmware" | sudo tee -a /etc/apt/sources.list.d/debian.list
```

`debian 13`

```
echo "deb https://deb.debian.org/debian/ trixie main contrib non-free non-free-firmware" | sudo tee -a /etc/apt/sources.list.d/debian.list
```

***Обновить конкретный пает пакета:***

```
sudo apt install --only-upgrade имя_пакета
```

***Проверка наличия пакете в репозитории***

```
sudo apt-cache policy имя_пакета
```

***Поиск установленных пакетов через apt***

```
sudo apt list --installed
```

```
sudo apt list --installed | grep python
```

```
sudo apt list --installed | grep python | wc -l
```
