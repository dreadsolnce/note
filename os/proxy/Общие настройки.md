Чтобы в терминале при использовании прокси, через команды export http_proxy=... и https_proxy=... данные переменные окружения были доступны и для команды sudo, необходимо в файле /etc/sudooers раскоментировать строку: `Defaults:%sudo env_keep += "http_proxy https_proxy ftp_proxy all_proxy no_proxy"`

```
sudo visudo
```

```
+++ Defaults:%sudo env_keep += "http_proxy https_proxy ftp_proxy all_proxy no_proxy"
```

#### Экспорт переменных окружения в bash:

```
export http_proxy=http://127.0.0.1:3080
export https_proxy=http://127.0.0.1:3080
```
