***Настройка доступа к серверу только по ssh ключам:***

Чтобы обезопасить себя от возможного брутфорса (взлома пароля простым перебором), 
необходимо отключить возможность такого входа.

```
sudo vim /etc/ssh/sshd_config
```

+++ PasswordAuthentication no

либо создать отдельный файл конфигурации:

```
sudo vim /etc/ssh/sshd_config.d/30-passwordAuth.conf
```

![[Снимок экрана от 2026-04-20 12-17-18.png]]

Проверка конфигурации sshd:

```
sudo sshd -t
```

```
sudo service sshd restart|reload
```
