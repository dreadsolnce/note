***Вывести все доступные пакеты***

````
zypper se 
````

***Вывести установленные пакеты***

```
zypper se -i
```

***Вывести информацию о пакете из определенного репозитория***

```
zypper search --repo home_SeaEagle1 nginx
```

***Вывести подробную информацию о пакете из указанного репозитория***

```
zypper info --repo home_SeaEagle1 nginx
```
