***Просмотр репозиториев***

список доступных (подключенных) репозиториев
```
zypper lr
```

Показать подробную информацию
```
zypper lr -d
```

Показать только включенные (активные) репозитории:
```
zypper lr -E
```

Показать URI/URL репозиториев:
```
zypper lr -u
```

***Обновить информацию о пакетах (синхронизировать метаданные репозиториев)*** 

```
sudo zypper ref
```

***Вывести все доступные пакеты***

````
zypper se 
````

***Вывести установленные пакеты***

```
zypper se -i
```

***Вывести информацию о пакете из определенного репозитория***

```
zypper search --repo home_SeaEagle1 nginx
```

***Вывести подробную информацию о пакете из указанного репозитория***

```
zypper info --repo home_SeaEagle1 nginx
```

***Установить пакет из указанного репозитория по псевдониму***

```
sudo zypper install --from <alias репозитория> <имя пакета>
```

```
sudo zypper install --from devel_languages_php_php85 php
```

***Удалить пакет***

```
sudo zypper remove php8 php8-cli
```

***Проверить состояние установленных пакетов на ошибки***

```
sudo zypper verify
```
